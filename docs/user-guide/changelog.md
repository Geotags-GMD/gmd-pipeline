# Changelogs

Changelogs of all GEMMA Plugin stable releases, which are also available [on GitHub](https://github.com/GMD-Repository/gemma-plugin/releases).

## [1.0.0] - 2026

### Highlights
- Added LGU CRS and Geometry Alignment Algorithm to Processing Provider
- Removed unnecessary subfolder and added README for better documentation
- Introduced QP Generation and Create Enumeration Areas QGIS Plugin
- Registered EA Delineation processing provider with UI actions and interactive preview support
- Improved Package Dialog functionality by solving existing bugs
- Harmonized legacy plugin references and introduced default presets for user convenience
- Enhanced drag-and-drop support for improved user experience
- Renamed GMD Pipeline to GEMMA and updated repository URLs
- Implemented GitHub Actions workflow for automated GEMMA Plugin releases
- Initialized VitePress documentation site with tool guides and updated README
